# Project Identity: Remedly

- **Vision:** Remedly is a simple, ultra-fast, and secure platform to buy crypto easily (fiat-to-crypto onramp).
- **Core Technology:** Uses Privy for simple, 100% non-custodial wallets (no seed phrases, email/social login). Uses Transak and MoonPay for buying crypto with credit card or Apple Pay. Uses Supabase for data.
- **What it is NOT:** Remedly is NOT a P2P marketplace. It has no peer-to-peer ads ("annonces") and no public user ratings/profiles. Do not suggest P2P features.
- **Primary Goal:** Removing the complexity of crypto for beginners while providing banking-grade security.
